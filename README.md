# Syntecxhub Data Science Internship Projects

## Projects Completed

### Project 1 — NumPy Data Explorer
- Array creation, indexing, slicing
- Mathematical and statistical operations
- Reshaping and broadcasting
- Save/load operations
- NumPy vs Python list performance comparison

### Project 2 — Pandas CSV Reader & Basic Analysis
- Read CSV/Excel into DataFrame
- Summary statistics (mean, median, min, max)
- Filter rows, select columns, slice subsets
- Save filtered results to CSV and Excel

### Project 3 — Data Cleaning Utility
- Detect and handle missing values (drop/fill/impute)
- Fix incorrect dtypes and parse dates (multiple formats)
- Remove duplicates and standardize column names
- Outputs cleaned dataset + cleaning log

## How to Run
1. Install dependencies: pip install numpy pandas openpyxl
2. Navigate to any project folder
3. Run: python filename.py
